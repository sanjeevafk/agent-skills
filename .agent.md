# agent-skills

**Context:** Global repository of reusable Claude Code skills  
**For:** Skill development, infrastructure maintenance, security scanning

## Quick Start

1. Review `CONTRIBUTING.md` for submission process
2. Use `/gsd-do` for GSD workflow tasks
3. Run Tirith scan before PR: `npx @agent-skills/tirith scan skills/my-skill/`
4. All skills tested + linted before merge

## File Structure

- `skills/` — Skill implementation (400+)
- `docs/` — Contributing guides, setup, architecture decisions
- `scripts/` — Build/install/sync utilities
- `security/` — Tirith configuration + policies

## Standards

- No `any` in TypeScript; strict mode required
- All changes tested + committed with Conventional Commits
- Security reviewed before merge (Tirith + manual check)
- No emojis in response generation
